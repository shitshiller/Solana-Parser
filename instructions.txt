Парсер токенов, находящихся у топ холдеров анализируемого мемкоина

Перед запуском убедитесь что у вас установленны следующие библиотеки Python:
> requests
> math
> json
> time
> matplotlib

Настройки парсера задаются в файле 'config.py':
TOKEN_HASH (строка) - Адрес контракта, анализируемого мемкоина
MIN_TOKEN_VALUE (цел. число) -  Минимальная сумма токена в долларах ($), чтобы токен в кошельке был включен в анализ.
WALLETS_AMOUNT (цел. число) - Количество кошельков, которые должны быть проанализированы среди держателей анализируемого мемкоина.
TOP (цел. число) - Количество токенов, которые должны быть отображены на итоговом графике по убыванию частоты встречаемости.

Чтобы запустить парсер, необходимо запустить скрипт 'main.py'
Во время работы в консоль будет выводиться текущее положение парсера
По окончанию работы в консоль будет выведено '--LOG: FILE COMPILED!' и открыто окно с графическим представлением данных
График можно сохранить в формате '.png', нажав иконку дискеты
По оканчанию работы будут созданы файлы, 'data.json', 'stat.json':
'data.json' - содержит статистику по каждому ТОП ХОЛДЕРУ, ссылка на solscan и токены, которые лежат на кошельке
'stat.json' - содержит статистику по каждому ТОКЕНУ, который лежит на кошельке ТОП ХОЛДЕРА, цена, частота встречаемости, холдеры

В папке 'data_example' представлены примеры работы на следующих настройках:
